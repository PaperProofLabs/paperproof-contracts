# governance

This directory is reserved for the PaperProof governance contracts.

The governance layer is intended to contain the on-chain logic for operational
authority, administrative continuity, capability control, and protocol
governance structures used by the PaperProof ecosystem.

Typical contents of this directory may include:

- a standalone Sui Move package for governance-related contracts
- source modules for administrative roles, authority transfer, and governance
  controls
- tests for operator succession and permission boundaries
- deployment or package notes specific to governance logic
