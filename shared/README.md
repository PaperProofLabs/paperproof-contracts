# shared

This directory is reserved for shared PaperProof contract components.

The shared layer is intended to host reusable Move packages, common types,
shared utilities, helper modules, constants, and cross-package abstractions
used by multiple PaperProof protocol packages.

Typical contents of this directory may include:

- one or more standalone Sui Move packages for shared logic
- common data types and utility modules reused across protocol contracts
- shared tests or fixtures for package integration
- reference material for cross-package dependencies
