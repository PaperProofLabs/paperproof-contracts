# publishing

This directory is reserved for the PaperProof publishing contracts.

The publishing layer is intended to contain the core on-chain logic for paper
registration, publication finalization, artifact binding, version continuity,
and related publication-state operations in the PaperProof protocol.

Typical contents of this directory may include:

- a standalone Sui Move package for publication-related contracts
- source modules for paper identifiers, publication records, and version state
- tests for publication and verification workflows
- deployment or package notes specific to the publishing layer
