# comments

This directory is reserved for the PaperProof comments and interaction
contracts.

The comments layer is intended to support protocol-level social interaction
around published artifacts, including comment records, reply structures,
reaction mechanisms, and other verifiable engagement flows tied to PaperProof
content.

Typical contents of this directory may include:

- a standalone Sui Move package for comment and interaction contracts
- source modules for artifact-linked discussion records and interaction state
- tests for posting, referencing, and reading social-layer records
- deployment or package notes specific to the comments layer
